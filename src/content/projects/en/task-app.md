## Overview

A task management application designed and developed from the ground up. From wireframes in Figma to a fully functional web app.

## Design Process

Started with user interviews to understand pain points in existing tools. Created lo-fi wireframes, iterated on the UI, and built a design system with reusable components.

## Development

Built with Astro for the marketing pages and React islands for the interactive dashboard. TypeScript throughout for type safety, with a clean component architecture.

## Key Features

- Drag-and-drop task boards
- Real-time collaboration
- Custom workflow automation
- Dark mode support
