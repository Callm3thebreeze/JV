## Overview

A front-end weather dashboard that pulls data from multiple weather APIs and presents it in a clean, readable format.

## Technical Highlights

- Vanilla JavaScript with no framework dependencies
- Responsive design that works from mobile to desktop
- Geolocation API for automatic location detection
- Chart.js for temperature and precipitation visualizations
- Service Worker for offline cached data

## What I Learned

This project was an exercise in building a polished product with vanilla technologies. No frameworks, no build tools — just clean JS, HTML, and CSS.
