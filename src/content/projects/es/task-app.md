## Resumen

Una aplicación de gestión de tareas diseñada y desarrollada desde cero. Desde wireframes en Figma hasta una aplicación web completamente funcional.

## Proceso de Diseño

Comencé con entrevistas a usuarios para entender los puntos de dolor de las herramientas existentes. Creé wireframes de baja fidelidad, iteré sobre la UI y construí un sistema de diseño con componentes reutilizables.

## Desarrollo

Construida con Astro para las páginas de marketing e islands de React para el dashboard interactivo. TypeScript en todo el proyecto para seguridad de tipos, con una arquitectura de componentes limpia.

## Características Principales

- Tableros de tareas con drag-and-drop
- Colaboración en tiempo real
- Automatización de flujos de trabajo personalizados
- Soporte para modo oscuro
