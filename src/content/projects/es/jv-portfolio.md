## Resumen

Un dashboard meteorológico front-end que obtiene datos de múltiples APIs del tiempo y los presenta en un formato limpio y legible.

## Aspectos Técnicos Destacados

- JavaScript vanilla sin dependencias de frameworks
- Diseño responsive que funciona desde móvil hasta escritorio
- API de Geolocalización para detección automática de ubicación
- Chart.js para visualizaciones de temperatura y precipitación
- Service Worker para datos cacheados offline

## Lo Que Aprendí

Este proyecto fue un ejercicio de construcción de un producto pulido con tecnologías vanilla. Sin frameworks, sin herramientas de build — solo JS, HTML y CSS limpio.
